# Suggested Kit for Wifi kit 32 testing

## Kit content
The kit may contains following content in order to run the examples
1. Wifi kit 32 module
2. breadboard
3. led x 8
4. 300 ohm resistor x8 
4. piezo buzzer
5. wires
6. led matrix (spi bus drived with MAX7219) x 4
